# दादाजी — Live Captions PWA files

Upload this whole folder with your app. It includes `index.html`, `manifest.webmanifest`, and PWA icons under `assets/icons/`.

If you only need to add these to an existing hosted app, copy `manifest.webmanifest` and the full `assets/icons` folder beside your `index.html`, then make sure the `<head>` contains the manifest and icon links.
